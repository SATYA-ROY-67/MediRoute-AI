# 🚑 MediRoute AI – Intelligent Emergency Healthcare Management System

An AI-powered emergency healthcare platform designed to reduce emergency response time by integrating patients, hospitals, ambulances, and blood banks into a single intelligent ecosystem.

## 📌 Features

- 🔐 Secure Authentication (Email & Google Login)
- 👤 Patient Dashboard
- 🏥 Hospital Dashboard
- 🚑 Ambulance Dashboard
- 🩸 Blood Bank Dashboard
- 👨‍💼 Admin Dashboard
- 🧠 AI Medical Triage
- 🚨 Emergency SOS System
- 🗺️ Live Emergency Map
- 📍 Dijkstra Shortest Path Visualization
- 📅 Appointment Management
- 📋 Medical Records Management
- 📞 Emergency Contacts
- 🔔 Realtime Notifications
- 📄 AI Report PDF Generation
- ☁️ Supabase Backend with Realtime Database

---

# 🛠 Tech Stack

### Frontend
- React
- TypeScript
- TanStack Router
- Tailwind CSS
- Shadcn UI

### Backend
- Supabase
- PostgreSQL
- Row Level Security (RLS)
- Realtime Database

### AI
- Lovable AI Gateway
- Gemini 2.5 Flash

### Algorithms
- Graph Data Structure
- Dijkstra's Shortest Path Algorithm
- Priority Queue (Min Heap)
